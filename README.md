# ゲーム開発ナレッジベース

プロジェクトを跨いで共有する、ゲーム開発の設定・考え方・事例・調査の蓄積。

## 使い方

- 何か知見が溜まったら、下記カテゴリの該当フォルダに 1 トピック 1 ファイルで追記する。
- ファイル名は内容がわかる kebab-case（例: `frame-rate-independent-movement.md`）。
- このトップ INDEX に 1 行リンクを足す。
- 横断的に関連するものは各ファイル内で相互リンクする。

## カテゴリ

| フォルダ | 内容 |
|---|---|
| [philosophy/](philosophy/) | 考え方・設計指針・判断基準 |
| [settings/](settings/) | プロジェクト共通の設定・環境・規約 |
| [success/](success/) | うまくいった事例・パターン |
| [issues/](issues/) | 不具合事例と原因・対処 |
| [research/](research/) | 調査内容・技術メモ・比較検討 |
