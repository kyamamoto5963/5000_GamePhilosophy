# 画像生成AI サービス調査（research/画像生成AI）

ゲームアセットを生成 AI で作るための、サービスごとの調査メモを貯める場所。
「何ができて・いくらで・どう連携できるか」と「本作で使えるか」の判断材料を残す。

## 索引

- [pixellab.md](pixellab.md) — ピクセルアート特化。キャラ/タイルセット/アニメ生成、Aseprite拡張・API・MCP対応

## 書き方ルール

- 1 サービス 1 ファイル。ファイル名はサービス名（小文字、例: `pixellab.md`）。
- 流れ: 何者か → できること → 料金 → 連携手段 → 本作での使いどころ・所感 → 出典・最終確認日。
- 料金・上限は条件込みで書く（プラン名・解像度上限・課金単位）。推測と実測（公式記載）は混ぜない。
- 日付は絶対日付。ファイルを足したらこの索引にも 1 行足す。
